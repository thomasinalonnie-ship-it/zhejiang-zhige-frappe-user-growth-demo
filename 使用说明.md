## 运行前提

该项目不是静态网页，不能直接双击 HTML 打开。

这是一个 Frappe v15 自定义 App，需要在已有 Frappe bench 环境中安装运行。

## 安装命令

在已有 Frappe v15 bench 中执行：

```bash
cd $PATH_TO_YOUR_BENCH
bench get-app ./path/to/user_growth_app
bench --site $SITE_NAME install-app user_growth_app
bench --site $SITE_NAME migrate
bench --site $SITE_NAME execute user_growth_app.api.seed_mock_data --kwargs '{"records": 90}'
bench start
```

## 建议验收顺序

1. 打开用户增长大屏，确认 KPI、筛选器、趋势图、地区分布和最近记录正常显示。
2. 检查浙江智格 logo 是否为透明 PNG，不应出现白色底图、白色卡片或白色边框容器。
3. 点击大屏按钮：`应用筛选`、`重置筛选`、`查看单据`、`查看报表`、`重置数据`。
4. 进入单据列表，确认筛选区、左侧侧栏、表格和 `新增用户服务单` 按钮没有遮挡。
5. 进入报表页，确认摘要、图表、表格和筛选器均为中文并可展示增长/流失数据。
